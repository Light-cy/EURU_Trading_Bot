🔄 LINUX → WINDOWS (MT5 Switching)
════════════════════════════════════════════════════════════════════════════════

Current Setup (Linux):
  MOCK_MT5=true     → Uses MockMT5 (demo data)

Switch to Real MT5 (Windows):
  Step 1: Update .env
          MOCK_MT5=false
  
  Step 2: Install MetaTrader5
          pip install MetaTrader5
  
  Step 3: Done! ✅
          System auto-detects and switches

Why it's easy?
  • TradeExecutor uses generic MT5 interface
  • Both MockMT5 and real MT5 have same methods
  • _init_mt5() handles the logic
  • Rest of code unchanged
