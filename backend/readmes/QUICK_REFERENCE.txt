⚙️  CONFIGURATION (.env)
───────────────────────────────────────────────────────────────────────────────
PAIR=EURUSD                 # Trading pair
TIMEFRAME=M15               # Candle timeframe
RISK_PERCENT=1.0            # Risk per trade
MIN_CONFIDENCE=85.0         # Signal threshold
MAX_OPEN_TRADES=3           # Max concurrent
MOCK_MT5=true               # true=Linux, false=Windows
FLASK_PORT=5000


🔄 SWITCH TO REAL MT5 (Windows)
───────────────────────────────────────────────────────────────────────────────
1. Change MOCK_MT5=false in .env
2. pip install MetaTrader5
3. Run on Windows
4. Done! ✅

🛠️  TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────
Server not starting?
  → Check port 5000 is free
  → Check Python installed: python --version
  → Install deps: pip install -r requirements.txt

No positions appearing?
  → Run: curl -X POST http://localhost:5000/api/analyze
  → Check confidence threshold in .env

Database issues?
  → Delete: rm trading_system.db
  → Will recreate on next run

MT5 won't connect?
  → On Linux: MOCK_MT5=true (default)
  → On Windows: MOCK_MT5=false + pip install MetaTrader5
