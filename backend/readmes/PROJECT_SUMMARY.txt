🔧 CONFIGURATION
════════════════════════════════════════════════════════════════════════════════

Edit .env file:

PAIR=EURUSD                 # Trading pair
TIMEFRAME=M15               # Candle timeframe (15 minutes)

RISK_PERCENT=1.0            # 1% risk per trade
MIN_CONFIDENCE=85.0         # Signals must be 85%+ confident
MAX_DAILY_LOSS=3.0          # Stop trading if down 3%
MAX_OPEN_TRADES=3           # Max 3 concurrent trades

MOCK_MT5=true               # true = Linux (MockMT5)
                            # false = Windows (Real MT5)

ENV=development             # development or production
FLASK_PORT=5000             # Server port
FLASK_DEBUG=true            # Debug mode

🎯 TRADING FLOW (AUTOMATED)
════════════════════════════════════════════════════════════════════════════════

Step 1: Fetch Market Data (1s)
  └─ MockMT5 generates or MT5 fetches 500 candles

Step 2: Technical Indicators (2s)
  ├─ RSI(14), EMA(20/50), MACD, ATR(14)
  ├─ VWAP, ADX(14), Bollinger Bands(20,2)
  └─ Each returns: signal + confidence

Step 3: SMC Analysis (1s)
  ├─ Market Structure (BOS, CHoCH, Trend)
  ├─ Order Block detection
  └─ Fair Value Gap detection

Step 4: Signal Aggregation (1s)
  ├─ Combine 10+ signals
  ├─ 60% consensus voting
  ├─ Weight by confidence
  └─ Final decision: BUY/SELL/NEUTRAL

Step 5: Risk Calculation (1s)
  ├─ Entry price = current price
  ├─ Stop loss = entry - 50 pips (configurable)
  ├─ Take profit = entry + 100 pips (2:1 RRR)
  ├─ Position size = 1% risk / risk pips
  └─ RRR = take profit / stop loss

Step 6: Validation (1s)
  ├─ Check max open trades
  ├─ Check daily loss limit
  ├─ Validate SL distance (10-500 pips)
  └─ Decision: Execute or skip

Step 7: Trade Execution (1s)
  ├─ Create order (BUY or SELL)
  ├─ Track position
  ├─ Monitor SL/TP
  └─ Calculate P&L

Step 8: Database Logging (1s)
  ├─ Store trade to database
  ├─ Log signal with details
  └─ Update account stats

Total: ~9 seconds per cycle ⚡
